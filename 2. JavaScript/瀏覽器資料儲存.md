# LocalStorage
最推薦也最簡單的資料儲存方法。

- 形式：key → value
    - `key` 跟 `value` 必須是字串，如果是 `<object>` 或 `<number>` 會自動轉換成 `<string>`
    - 存取 JSON 格式要先經過轉換。
- 生命週期：沒有過期時間，除非手動刪除
    - 意思是說關掉分頁、關掉瀏覽器再打開也都還在。
- 不像 Cookie 會隨著 Request 送到 Server 端，Storage 只作用在 Client 端瀏覽器
    - 適合儲存較複雜且不敏感的資料 ( 喜好設定、顏色樣式等等 )
